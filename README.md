 # Changelog

Tous les changements de Naviwest.Mobile seront documentés ici.

---

## [1.2.0 (23)] - 10/12/2025

### Ajouté
- Suppression d'une ligne de feuille de temps
- Suppression d'une ligne de charge
- Filtrer l'affichage des ressources bloquées
- Gestion des matériels
- Affichage des montants des primes et matériels

### Modifié
- Correction du plantage de l'appli lors de la sélection d'une charge
- Gestion des erreurs pour toutes les communications avec Naviwest
- Désormais les jours avec une quantité égale à 0 ne sont plus sauvegardés dans Naviwest (26.2.0.4)
- Ré-activation des logs naviwest (26.2.0.5)
- Rafraichissement visuel du coût total d'une ligne de charge (26.2.0.5)

### Note
- Nécéssite au minimum la version 26.2.0.3 de l'extension Business central
